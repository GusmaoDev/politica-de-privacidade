Política de Privacidade

Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações processadas por nosso aplicativo de automação de mensagens via WhatsApp, integrado à plataforma n8n.

1. Coleta de Dados

O aplicativo pode processar os seguintes dados:

  • Nome e número de telefone de contatos que interagem via WhatsApp
  • Mensagens automatizadas enviadas via integração

Esses dados são obtidos por meio de integrações voluntárias com outras plataformas, como Hotmart ou sistemas de CRM.

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Envio de mensagens automáticas via WhatsApp
  • Execução de fluxos de automação no n8n

3. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos dados pessoais com terceiros. Os dados são usados exclusivamente para as finalidades operacionais da automação.

4. Armazenamento e Segurança

Os dados são armazenados com segurança nas plataformas utilizadas, incluindo servidores auto-hospedados do n8n e ferramentas integradas. Adotamos boas práticas de segurança para proteger essas informações.

5. Direitos do Usuário

O usuário pode solicitar, a qualquer momento, a alteração ou exclusão dos seus dados, entrando em contato conosco através do e-mail informado abaixo.

6. Contato

Para dúvidas, solicitações ou mais informações sobre esta política, entre em contato:

  • E-mail: contato@seudominio.com
